# Timetable Conflict Detector

A Java Servlet-based web application that detects timetable conflicts for faculty and classrooms.

## Technologies Used
- Java (Servlet & JSP)
- JDBC
- MySQL
- Apache Tomcat
- NetBeans IDE

## Features
- Faculty login & registration
- Add / Update / Delete timetable
- Conflict detection logic
- Admin dashboard
- Classroom & Subject management

## How to Run
1. Import project into NetBeans
2. Configure MySQL database
3. Deploy on Apache Tomcat
4. Run on localhost
