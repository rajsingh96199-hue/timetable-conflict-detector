# Timetable Conflict Detector

A Java-based web application designed to detect and prevent scheduling conflicts for faculty and classrooms.

## Tech Stack

- Java (Servlet & JSP)
- JDBC
- MySQL
- Apache Tomcat
- NetBeans (Ant project)

## Features

- Faculty Registration & Login
- Add / Update / Delete Timetable
- Conflict Detection Algorithm
- Admin Dashboard
- Classroom Management
- Subject Management

## System Architecture

Three-tier architecture:
Presentation Layer → Servlet Layer → Database Layer

## How to Run

1. Clone the repository
2. Import into NetBeans
3. Configure MySQL database
4. Deploy using Apache Tomcat
